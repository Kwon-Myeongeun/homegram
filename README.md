### 대화 카드

한 집에 있어도 대화가 없는 우리 가족, 대화를 이어가고 싶지만 무슨 말을 해야 할지 모르겠는 당신을 위해 매일 대화 카드를 보내드려요. 매일 업데이트 되는 대화 카트를 통해 나의 소중한 사람들과 매일 대화를 나누어 보세요.
